# Pan Lab Exercise Coach SMS Opt-In

Pan Lab Exercise Coach is an SMS-based participant support program operated by Justin Reynolds, a sole proprietor.

This page shows the complete user-facing opt-in flow for the Pan Lab Exercise Coach SMS program. It is intended to demonstrate exactly what a participant sees before consenting to recurring text messages. This page itself is the public opt-in example referenced in toll-free verification materials, and this page is where end users learn the `JOIN` keyword and toll-free number before initiating consent by text.

## How To Join

To join the Pan Lab Exercise Coach SMS program:

1. Read the opt-in disclosure below.
2. Text `JOIN` to `+1 (855) 906-1132`.
3. Reply `YES` to the confirmation message to complete enrollment.

No login or web form is required. Enrollment is completed through SMS so the program can stay phone-first and message-based.

Because this is a text-message opt-in workflow, there is no web form or phone-number entry field. The participant opts in by sending `JOIN` from the mobile number they want enrolled, and then confirms consent by replying `YES`.

## Opt-In Disclosure

Text `JOIN` to `+1 (855) 906-1132` to enroll in the Pan Lab Exercise Coach SMS program.

Participation is optional.

By texting `JOIN`, you are requesting recurring SMS messages related to the Pan Lab Exercise Coach program, including onboarding, reminders, check-ins, follow-ups, and participant support messages.

Message frequency varies. Message and data rates may apply. Reply `HELP` for help or `STOP` to opt out at any time.

## Full Opt-In Example

This is the complete expected opt-in workflow for a participant:

### Step 1: Public Opt-In Call To Action

The participant sees the following instruction on this public opt-in page on the Pan Lab website:

> Pan Lab Exercise Coach  
> Text `JOIN` to `+1 (855) 906-1132` to enroll in SMS coaching messages for onboarding, reminders, check-ins, and follow-ups. Message frequency varies. Message and data rates may apply. Reply `HELP` for help or `STOP` to opt out.

### Step 2: User Initiates Opt-In By Text

The participant sends:

> `JOIN`

### Step 3: SMS Confirmation Message

The participant receives this confirmation text:

> Pan Lab Exercise Coach: Reply `YES` to confirm that you want SMS messages about onboarding, reminders, check-ins, and follow-ups for the Pan Lab Exercise Coach program. Message frequency varies. Message and data rates may apply. Reply `HELP` for help or `STOP` to opt out.

### Step 4: User Confirms Consent

User sends:

> `YES`

### Step 5: Enrollment Confirmation

> Pan Lab Exercise Coach: You are now enrolled in Pan Lab Exercise Coach SMS messages. You can reply with questions at any time. Message frequency varies. Message and data rates may apply. Reply `HELP` for help or `STOP` to opt out.

Participants who opt in may receive messages such as:

These messages are for participant support and program operations. They are not sold to third parties.
